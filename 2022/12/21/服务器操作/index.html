服务器操作

Author: lufei Date: 十二月 21, 2022  12:12:52

服务器基础操作

切换目录

cd /usr/local
###远程下载linux系统mysql压缩包（中间可替换版本号）
wget https://dev.mysql.com/get/Downloads/MySQL-5.7/mysql-5.7.26-linux-glibc2.12-x86_64.tar.gz

解压mysql压缩包

tar -zxvf mysql-5.7.26-linux-glibc2.12-x86_64.tar.gz

将解压后的文件命名为 mysql

mv mysql-5.7.26-linux-glibc2.12-x86_64 mysql

创建mysql用户组

groupadd mysql

向用户组中添加mysql用户

useradd -r -g mysql mysql

切换到mysql文件下并创建数据文件

cd /usr/local/mysql
mkdir data

切换目录 指定用户和用户组

cd /usr/local/
chown -R mysql mysql/
chgrp -R mysql mysql/
###初始化mysql
cd /usr/local/mysql/bin
yum -y install numactl
./mysqld –initialize –user=mysql –basedir=/usr/local/mysql/ –datadir=/usr/local/mysql/data/ –lc_messages_dir=/usr/local/mysql/share –lc_messages=en_US
（注意：记得初始化的mysql登录密码 root@localhost: 初始化密码，如果忘记密码或者想重新初始化，可以先将mysql/data目录中文件删除，然后再执行初始化命令）

配置my.cnf

vi /etc/my.cnf

添加如下配置

[client]
port = 3306
default-character-set=utf8

[mysqld]
basedir=/usr/local/mysql/
datadir=/usr/local/mysql/data/
port = 3306
character-set-server=utf8
default_storage_engine = InnoDB
lower_case_table_names = 1
sql_mode=STRICT_TRANS_TABLES,NO_ZERO_IN_DATE,NO_ZERO_DATE,ERROR_FOR_DIVISION_BY_ZERO,NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION

启动MySQL

cd /usr/local/mysql/bin
./mysqld_safe –user=mysql &

设置开机自启动

cp /usr/local/mysql/support-files/mysql.server /etc/init.d/mysql
vi /etc/init.d/mysql

修改为如下配置

basedir=/usr/local/mysql/
datadir=/usr/local/mysql/data/

授权

chmod +x /etc/init.d/mysql

设为开机启动

chkconfig –add mysql

开启mysql服务

启动服务：service mysql start
停止服务：service mysql stop
重启服务：service mysql restart
查看服务：service mysql status

登录MySQL

cd /usr/local/mysql/bin
./mysql -u root -p
（注意：此时需要用到上面的初始化密码）

修改密码，登录授权

修改密码：set password=password(“123456”);将密码更改为root
登录授权：grant all privileges on . to’root’ @’%’ identified by ‘123456’;
授权生效：flush privileges;

开启 mysql -u root -p 启动mysql

ln -s /usr/local/mysql/bin/mysql /usr/bin

#防火墙 /etc/sysconfig/iptables文件
*filter
:INPUT ACCEPT [0:0]
:FORWARD ACCEPT [0:0]
:OUTPUT ACCEPT [0:0]
-A INPUT -m state –state ESTABLISHED,RELATED -j ACCEPT
-A INPUT -p icmp -j ACCPET
-A INPUT -i lo -j ACCEPT
-A INPUT -m state –state NEW -m tcp -p tcp –dport 22 -j ACCEPt
-A RH-Firewall-1-INPUT -m state –state NEW -m tcp -p tcp –dport 3306 -j ACCEPt
-A INPUT -j REJECT –reject-with icmp-host-prohibited
-A FORWARD -j REJECT –reject-with icmp-host-prohibited
COMMIT

cd /usr/local/mysql/bin

查看当前防火墙状态。

systemctl status firewalld

关闭当前防火墙。

systemctl stop firewalld

开机防火墙不启动。

systemctl disable firewalld

linux基础指令

  1. cd：改变目录
  2. cd .. ：回退到上一个目录，直接cd进入默认目录（根目录）
  3. pwd：显示当前目录路径
  4. ls(ll)：列出当前目录中的所有文件，只不过 ll （两个L）列出的更详细
  5. touch：新建一个文件，如touch index.js会在当前目录下创建一个index.js文件
  6. rm：删除一个文件，rm index.js会把当前目录下的index.js文件删除
  7. mkdir：新建一个目录（文件夹）
  8. rm -r：删除一个文件夹，rm -r main就是删除当前目录下的main文件夹
  9. mv：移动文件，mv index.js webRoot把当前目录下的index.js文件移动到webRoot目录下（必须同目录下）
  10. reset：重新初始化终端/清屏
  11. clear：清屏
  12. history：查看历史命令
  13. help：帮助
  14. exit：退出
  15. #表示注释
  16. vim 表示进入当前文件

rmdir：删除空文件（文件非空时不可删除）
cp：复制cp /home/mian/* /home/webRoot 复制main下所有文件到webRoot下
